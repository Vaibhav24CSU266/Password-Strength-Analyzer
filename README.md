# Password Strength Analyzer

This project implements a Password Strength Analyzer using Python. It evaluates the strength of a user-entered password based on security best practices and provides suggestions to create a stronger password.

## Features
- Check password length
- Detect uppercase and lowercase letters
- Check for numbers and special characters
- Evaluate password strength (Weak, Medium, Strong, Very Strong)
- Detect common passwords
- Prevent reuse of old passwords (simulated database)
- Suggest a stronger password if the entered password is weak

## Technologies Used
- Python
- Regular Expressions (re module)
- Random
- String

## How to Run

```bash
python password_strength_analyzer.py
```
